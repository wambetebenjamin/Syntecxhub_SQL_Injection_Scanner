# SQL Injection Scanner

This project is part of Week 3 Cybersecurity Internship.

## Description
A Python-based SQL Injection Scanner that:
- Sends crafted payloads
- Detects SQL injection indicators
- Uses authentication cookies
- Logs vulnerabilities
- Implements concurrency

## Technologies Used
- Python
- Requests
- Colorama
- ThreadPoolExecutor

## Testing Environment
- DVWA (Damn Vulnerable Web Application)
- XAMPP (Apache + MySQL)

This tool is for educational purposes only.
Tested only on permitted local environments.
